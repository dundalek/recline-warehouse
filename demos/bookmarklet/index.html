DataExplorer bookmarklet

Lets suppose you found a web page, which has useful data presented in a static table.
Using this script you can transform the table to a rich widget, which enables sorting, filtering, graphing, visualizing on the map, etc.

Try it by clicking Test

You can use it as a bookmarklet on any web page.
Drag the following into the bookmark bar for later usage: DataExplorer bookmarklet
